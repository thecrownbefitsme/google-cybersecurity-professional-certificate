# Security Incident Report Sample

## Incident Summary
A potentially malicious file execution was detected by the antivirus software on an employee’s workstation.

## Date Detected
March 2026

## Affected System
Employee Windows workstation connected to corporate network.

## Detection Method
Antivirus alert and unusual system behavior.

## Actions Taken
- Isolated the system from the network
- Ran a full antivirus scan
- Reviewed system logs
- Removed the malicious file
- Reset user credentials

## Recommendation
- Educate staff on safe software installation
- Enforce least privilege policy
- Monitor endpoints with EDR tools
