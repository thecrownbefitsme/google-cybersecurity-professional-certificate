# tcpdump Packet Capture Analysis

I used tcpdump to capture live network packets and analyze traffic.

## Command Used

sudo tcpdump -i any

## Observations

- Observed DNS requests to external servers
- Identified HTTP traffic between local machine and web servers
- Saw TCP handshake process (SYN, SYN-ACK, ACK)
- Noticed source and destination IP addresses
- Learned how to stop capture with Ctrl + C

## Learning Outcome

This exercise helped me understand how data travels across a network and how security analysts can monitor suspicious traffic using packet capture tools.
