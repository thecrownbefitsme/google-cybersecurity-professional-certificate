# Linux Fundamentals for Security

Linux is widely used in cybersecurity for server management, log analysis, and security monitoring.

## Important Commands

- pwd – show current directory
- ls – list files
- cd – change directory
- grep – search inside files
- chmod – change permissions
- ps aux – view running processes
- kill – terminate process

## Why Linux Matters in Security

Security analysts use Linux to:
- Analyze logs
- Monitor processes
- Investigate suspicious activity
- Work with security tools
