# Networking Basics for Cybersecurity

Understanding networking is essential to detect malicious traffic.

## Core Concepts

- IP Address – identifies devices
- DNS – resolves domain names
- TCP/UDP – communication protocols
- HTTP/HTTPS – web communication
- Ports – communication endpoints

## Why It Matters

Security analysts monitor network traffic to detect:
- Unauthorized access
- Data exfiltration
- Malware communication
